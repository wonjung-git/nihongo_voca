# NEW どきどき(도키도키) 일본어 초급 上 단어 연습 웹 사이트

## 만든 목적
일본어 교재로 공부할 때 단어를 외우는 과정이 가장 중요하면서도 반복이 필요한 작업입니다. 
종이 단어장이나 단순 암기 방식보다 랜덤 플래시카드 방식이 단어 기억에 더 효과적이라는 점에서, 교재 단원별로 단어를 골라 바로 연습할 수 있는 도구가 필요해 이 사이트를 만들었습니다.

특히 NEW どきどき(도키도키) 일본어 초급 교재를 사용하는 학습자들이 시험 범위에 맞춰 단어를 효율적으로 준비할 수 있도록 설계했습니다.

## 주요 기능
#### 단원별 선택
시험 범위에 맞게 특정 과만 골라서 학습할 수 있습니다.
#### 랜덤 순서
단어가 매번 다른 순서로 나와 외울 때 위치에 의존하지 않게 됩니다.
#### 보기 항목 선택
한자·가나·한글 중 먼저 볼 항목을 자유롭게 선택할 수 있습니다.
#### 모바일 최적화
스마트폰으로도 편리하게 단어를 학습할 수 있습니다.
